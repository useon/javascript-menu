# 미션 - 점심 메뉴 추천

[woowacourse-precourse/javascript-menu](https://github.com/useon/javascript-menu)
해당 문서의 기능 요구 사항, 프로그램 요구 사항, 과제 진행 사항을 따른다.

### 🎮 프로그램 동작 순서

1. 점심 메뉴 추천 프로그램 시작 문구를 출력한다.
2. 카테고리를 생성한다.
3. 코치의 이름을 입력받는다.
4. 각 코치가 못 먹는 메뉴를 입력받는다.
5. 각 코치별로 메뉴를 추천한다.
6. 메뉴 추천 결과를 출력한다.
7. 추천 완료 문구를 출력한다.
